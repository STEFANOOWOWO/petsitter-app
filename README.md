# PetSitter 🐾
Descripción del proyecto
# ¿Qué es PetSitter?
PetSitter es una plataforma web que conecta dueños de mascotas con cuidadores de confianza. Permite encontrar, contactar y contratar cuidadores para perros y gatos de forma rápida y sencilla.
# ¿Qué problema intenta resolver?
Muchas personas no tienen con quién dejar a su mascota cuando salen de viaje o trabajan. PetSitter resuelve ese problema conectando dueños con cuidadores verificados cercanos.
# ¿Quiénes son los usuarios principales?
Dueños de mascotas que necesitan un cuidador, y cuidadores que ofrecen sus servicios.

# Funcionalidades actuales
- Landing principal con información del servicio y testimonios de usuarios.
- Inicio de sesión para cuidadores con validación de campos.
- Registro de cuidadores con selección de tipo de mascota y experiencia.
